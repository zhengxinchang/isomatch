Group 3: Sequence-derived QC attributes
Input required: reference genome FASTA (indexed, mmap-accessible), optional polyA motif list file.

all_canonical: bool
对 query transcript, 读取Txtype是否是ALL_CA

_rts_stage: bool
对 query transcript 的每个 junction，检测该 junction 是否可能由 reverse transcriptase template switching 产生。
算法：取 junction donor 上游的 exonic 序列（末端 20-30bp）和 acceptor 下游 intron 起始序列（前 20-30bp），计算两段序列之间的短 repeat 匹配度。如果在 junction 边界检测到 ≥ 某阈值（如连续 ≥4bp 的 direct repeat），标记该 junction 为 RTS suspect。
任一 junction 被标记则整个 transcript _rts_stage = true，否则 false。Mono-exon 转录本设为 false。

perc_a_downstream_tts: f32
确定 query transcript 的 TTS 基因组坐标：+ strand 取最后一个 exon 的 end，- strand 取第一个 exon 的 start。
从 TTS 开始，沿转录方向的下游提取 20bp 基因组序列。+ strand 取 genome[TTS..TTS+20]，- strand 取 genome[TTS-20..TTS] 的反向互补。
计算该 20bp 窗口中 A 碱基的占比：count('A') / 20.0 * 100.0。值域 0.0–100.0。

seq_a_downstream_tts: Option<String>
与 perc_a_downstream_tts 同步计算。保存上述提取的 20bp 基因组序列原始字符串（已 strand-correct，即 - strand 已取反向互补）。
如果 TTS 位于染色体末端不足 20bp 可提取，设为 None。

_poly_a_motif: Option<String>
额外输入： 一个排序好的 polyA motif 列表文件（如 AATAAA, ATTAAA, AGTAAA, ...），按生物学优先级从高到低排列。
从 TTS 上游 50bp 提取基因组序列（strand-corrected）。+ strand 取 genome[TTS-50..TTS]，- strand 取 genome[TTS..TTS+50] 的反向互补。
按 motif 列表顺序，在该 50bp 序列中搜索每个六聚体。返回第一个命中的 motif 字符串（如 Some("AATAAA")）。如果全部未命中，返回 None。

_poly_a_motif_found: bool
_poly_a_motif.is_some() 的简化。找到任何 motif 则 true，否则 false。

_poly_a_dist: Option<i32>
与 _poly_a_motif 同步计算。如果找到了 motif，记录该 motif 最后一个碱基在 50bp 窗口中相对于 TTS（position 0）的位置。
计算方式：motif 最后一个碱基在基因组上的坐标为 motif_end，则 _poly_a_dist = motif_end - TTS。因为 motif 在 TTS 上游，该值始终 ≤ 0。典型值在 -10 到 -30 之间，表示 canonical polyadenylation signal 位置。
如果 _poly_a_motif 为 None，则此字段也为 None。

Group 4: Orthogonal evidence
Input required: optional CAGE peak BED file, optional polyA site BED file. 如果用户未提供对应 BED 文件，该组所有字段均为 None。

within_cage_peak: Option<bool>
确定 query transcript 的 TSS 基因组坐标：+ strand 取第一个 exon 的 start，- strand 取最后一个 exon 的 end。
在 CAGE peak BED 中，按 chromosome + strand 筛选，判断 TSS 是否落在任一 BED 区间 [start, end] 内。落在区间内则 Some(true)，否则 Some(false)。未提供 BED 文件则 None。
实现：对 CAGE BED 按 chromosome + strand 构建 COITree，对 TSS 做 point overlap query。

dist_to_cage_peak: Option<i32>
在 TSS 所在 chromosome + strand 上，搜索 TSS 上下游 10000bp 范围内最近的 CAGE peak。
计算 TSS 到该 peak 中点 的有符号距离：dist = TSS - midpoint(peak)。+ strand 下负值表示 peak 在 TSS 上游（TSS 在 peak 下游），正值表示 peak 在 TSS 下游。- strand 取反。
如果 10000bp 窗口内无 peak，返回 None。未提供 BED 文件也返回 None。
实现：对 CAGE BED 按 chromosome + strand 构建 COITree，对区间 [TSS - 10000, TSS + 10000] 做 range query，取中点距离最小的 peak。

within_poly_a_peak: Option<bool>
确定 query transcript 的 TES 基因组坐标（同 perc_a_downstream_tts 中的定义）。
在 polyA site BED 中，按 chromosome + strand 筛选，判断 TES 是否落在任一 BED 区间 [start, end] 内。落在则 Some(true)，否则 Some(false)。未提供 BED 文件则 None。
实现：对 polyA BED 按 chromosome + strand 构建 COITree，对 TES 做 point overlap query。

dist_to_poly_a_site: Option<i32>
在 TES 所在 chromosome + strand 上，搜索 TES 上下游 10000bp 范围内最近的 polyA site。
计算 TES 到该 site 中点 的有符号距离：dist = TES - midpoint(site)。+ strand 下负值表示 site 在 TES 下游（转录本可能提前截断），正值表示 site 在 TES 上游。- strand 取反。
如果 10000bp 窗口内无 site，返回 None。未提供 BED 文件也返回 None。
实现：对 polyA BED 按 chromosome + strand 构建 COITree，对区间 [TES - 10000, TES + 10000] 做 range query，取中点距离最小的 site。
